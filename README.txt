Перед началом работы в "Информационной системе для обучения" следует установить приложения по ссылкам:
1.https://www.apachefriends.org/download.html
2.https://www.java.com/ru/download/

После установки запустите приложение XAMPP control Panel от имени администратора.
Дальше вы увидите Service Module PID(s) Port(s) Actions. Под словом Actions нажмите кнопку Start напротив слова MySQL.
После всех выполненных шагов запускайте приложение Information_System.

После открытия приложения Information_System создайте аккаунт. Чтобы создать аккаунт нажмите на слова Создать аккаунт.

При создании аккаунта запишите Кодовое слово, которое поможет в восстановлении пароля, если вы его забудете.

После создания аккаунта в открытом окне нажмите кнопку Обучения. Дальше нажмите кнопку Обучение, после нажмите кнопку Начать обучение, после выберите язык, который хотите начать изучать, дальше следуйте всем инструкциям, которые написаны в нём для того, чтобы начать обучения.

В данном приложении вы найдёте учебники, упражнения.

Удачи в обучении!
